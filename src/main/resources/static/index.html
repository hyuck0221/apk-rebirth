                    설치할 때 "호환되지 않음" 이나 "차단됨" 이 뜨면 펼쳐보세요
                    
    
                        
  1. 우회 옵션 켜기: 업로드 화면에서 "악성 앱으로 차단된 경우 우회 시도" 를 체크하고 다시 변환해 보세요.
    2. 
                        
  2. Play 프로텍트 끄기: Play 스토어 → 프로필 → Play 프로텍트 → 설정 → 기기 스캔 끄기. 설치 후 다시 켜도 돼요.
    3. 
                        
  3. "어쨌든 설치": 경고 창이 뜨면 상세 정보 → 설치(또는 무시하고 설치)를 눌러보세요.
    4. 
                        
  4. ADB로 강제 설치: PC에 연결해 adb install --bypass-low-target-sdk-block 파일.apk.
    5. 
                        
  5. 알 수 없는 출처 허용: 설정 → 앱 → 이 앱을 설치한 브라우저/파일관리자 → 알 수 없는 앱 설치 허용.
